Module Camera


module Camera: sig .. end
The camera module handles advanced camera animations

type camera_op =
| Translate of Gl.point3 * Gl.point3 (*Translate from to*)
| Rotate of float * float * Gl.vect3 (*Rotate angle_from angle_to vertex*)
A camera operation is a Translation or Rotation
type camera_op_list = camera_op list * float * (Transition.trans * Transition.ease)
Camera transitions are a list of camera operations with their duration (in frames) and transition and easing options
class camera_model : object .. end
The camera_model class applies advanced camera transformations before the data is rendered
