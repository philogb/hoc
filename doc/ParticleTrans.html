Module ParticleTrans


module ParticleTrans: sig .. end
This module provides multiple particle transformations. A particle transformation takes a frame as formal parameter and returns a frame. Custom frame transformations should be appended here

type transformation =
| Idle
| Project of float * float * float
| Random
Frame/Particle transformations should be defined as type constructors
val idle : 'a -> 'a
Idle transformation does nothing
val project : transformation ->
       VertexType.depth_vertex list -> VertexType.depth_vertex list
Project transformation projects all particles to a surface
val random : VertexType.depth_vertex list -> VertexType.depth_vertex list
Random transformation assigns a random position to each particle

       VertexType.depth_vertex list -> VertexType.depth_vertex list
